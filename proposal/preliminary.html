Preliminary Data Report¶

Preprocessing¶

The preprocessing was done in the integrate_and_clean.py file, which does a number of things. It was decided that integration would be done before the clean up because those data structures would give access to an easier way to do most of the cleaning functions

  1. Integrations a. Trailing spaces need to be removed for merging b. Merges the user ratings to have an imdbId column so that the movies are easier to identify c. Merges the several csv documents into a single dataframe so operations are simpler d. Exports as lz4 because it has a fast compression algorithm
  2. Cleanup a. Removes unnecessary columns - budget, use_gross_income, worldwide_gross_income, metascore. These are all missing too many of their rows to be worthwhile b. Remove lower outliers from reviews_from_users, reviews_from_critics, voters, and remove all the outliers from duration. c. Removing movies that don't exist - a user reviews a movie that has an id that doesn't exist

d = movie_df.groupby('duration')['imdbId'].nunique()
f = []

for value in d:
    f.insert(-1, value)

d = {'Run Time': movie_df['duration'].unique(), 'Frequency': f}
df = pd.DataFrame(data=d)

print("Scatter plot of the durations")
df.plot.scatter(stacked=True, x='Run Time', y='Frequency', color="r")
plt.show()

df['quantile'] = pd.qcut(df['Run Time'], q=10)

df = df.drop(['Run Time'], axis=1)

df = df.groupby('quantile').agg('sum')

print("Histogram of duration divided into 10 bins")
df.plot.bar(stacked=True, color="g")
plt.show()

df = movie_df
df = df.drop(df.columns.difference(['duration']), 1, inplace=False)

print("Box plot of the durations")
df.plot.box(stacked=True)
plt.show()
